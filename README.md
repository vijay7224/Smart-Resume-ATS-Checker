🚀 AI-Based ATS Resume Analyzer using NLP
📌 Overview

AI-Based ATS Resume Analyzer is a web application that evaluates resumes by comparing them with a job description. It calculates an ATS score, identifies missing keywords, and helps users improve their resumes for better job matching.

🎯 Features
📄 Upload Resume (PDF)
📝 Paste Job Description
🎯 ATS Score Calculation
🔍 Missing Keywords Detection
⚡ Fast & Simple UI (Flask-based)

🧠 How It Works
Extracts text from resume using PyPDF2
Processes text using NLP techniques
Converts text into vectors using TF-IDF
Calculates similarity using Cosine Similarity
Displays ATS score and missing keywords

🛠️ Tech Stack
Python
Flask
PyPDF2
Scikit-learn
HTML, CSS
